Welcome to YOLO-MultiSpectral

YOLO-MultiSpectral is an open-source deep learning framework for real-time multispectral object detection and instance segmentation.

🌟 Features

  • 4+ channel multispectral input support (e.g., RGB + NIR + RedEdge)
  • YOLOv8/YOLOv11-based backbone with CBAM and ECA attention
  • Precision agriculture and drone imagery optimized
  • Easy integration into QGIS and geospatial pipelines

🚀 Quick Start

See the GitHub Repository to get started or view the README.
