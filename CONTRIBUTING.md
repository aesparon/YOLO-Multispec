# Contributing to YOLO-MultiSpec

We welcome contributions to improve this multispectral detection framework!

## 🚀 How to Contribute

1. **Fork** the repository
2. **Clone** your fork:  
   `git clone https://github.com/your-username/YOLO-MultiSpec.git`
3. **Create a branch** for your fix or enhancement:  
   `git checkout -b feature-your-feature-name`
4. Make changes and **test** thoroughly
5. Commit your changes and **push**
6. Submit a **Pull Request** and describe what your PR solves

...

Thank you for helping make YOLO-MultiSpec better! 💡
