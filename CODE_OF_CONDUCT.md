# Code of Conduct

All contributors to YOLO-MultiSpec are expected to adhere to the following code of conduct.

## 💬 Be Respectful
Treat others with kindness and professionalism. Respect differing viewpoints.

## 🤝 Collaborate Openly
Be welcoming to newcomers and encourage inclusive participation.

## 🚫 Unacceptable Behavior
- Harassment or discrimination
- Disrespectful or abusive language
- Deliberate intimidation or threats

Violations may result in a ban from participation.

---

This code of conduct is adapted from the Contributor Covenant.
